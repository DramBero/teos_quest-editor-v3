# tride-v3

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Compile for Desktop

```sh
npm run tauri build
```

Stack: 
Vue3 (Composition API + TS)
Pinia - for storage
Tauri - for compiling into desktop apps
TresJS (ThreeJS) - for 3D model rendering
Dexie - for IndexedDB
